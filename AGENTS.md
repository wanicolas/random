# Random Magazine - Guide pour Agents IA

## Stack Technique

- **CMS**: Ghost v6
- **CSS Library**: TailwindCSS v4
- **Templating**: Handlebars
- **Package Manager**: npm

## Particularités

Ghost est utilisé en local via un docker-compose.yml.
Seul le thème custom est versionné et "push" en production quand nécessaire.
